POS All Orders List

This module is linked with POS Order Return , POS Order Reprint and Pos Order History & Re-Order

Features
  • This module is used to see the list of all the orders within a running POS.
  • POS User can also see the list of orders associated with individual customers.
  • POS Session View
  • Backend Configuration

POS All Orders List

Press the "All Orders" button to view the list of all the loaded orders.
User can also search the orders based on customer name as well as Order reference number.
User can view the orders associated with individual customers also.

Backend View And Configuration

These are the configuration settings which specify how many orders from the past would be loaded in POS when it starts.
